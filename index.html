FleetView lit et pilote ta flotte via l'API GitHub, directement depuis ce navigateur. Le token reste en localStorage — il ne transite jamais ailleurs que vers api.github.com.

  1. Crée un token fine-grained sur github.com/settings/personal-access-tokens
  2. Accès : tous les repos du compte Thibaud888
  3. Permissions : Contents, Issues, Pull requests, Actions (read & write) + Metadata (read) ; Administration seulement pour créer des repos d'ici.
  4. Expiration : No expiration — sinon il faudra revenir en recréer un régulièrement.
